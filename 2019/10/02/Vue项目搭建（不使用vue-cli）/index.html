一、创建项目结构

1.1、初始化项目

新建 package.json。

1.2、安装依赖

npm 安装 vue、vue-router、vue-loader、webpack、webpack-cli，因为这些只是开发环境所需要的依赖，所以使用 –save-dev 安装到开发环境（见 package.json devDependencies）下：

webpack 4.x 已经把 cli 单独拎出来了，所以还要安装 webpack-cli。然后因为 webpack 本身其实直接能处理的只有 js 资源，是通过各种 loader 让其他资源可以被 webpack 打包处理的。

要用 vue 写单文件组件（就是 .vue 文件），所以就还要安装 vue-loader。

1.3、创建目录、文件

  • src/assets 存放一些静态文件，例如图片、样式表等；
  • src/pages 存放组件；
  • src/router 路由配置；
  • src/App.vue 根组件；
  • src/main.js 入口文件，程序运行会依照里面的配置；
  • index.html 展现页面。
