LeanCloud使用

发表于 2019-01-02 | 分类于 Web

一、什么是 BaaS

在说 LeanCloud 之前，先了解一下 BaaS。

BaaS（后端即服务：Backend as a Service）公司为移动应用开发者提供整合云后端的边界服务。

如果你是一个前端开发，可以把这样的平台当做自己的服务器，来存储数据等。这样在不懂后端技术的情况下，依然可以请求数据，渲染到页面上，不必写入一堆假的据来学习一些技术和框架。我的理解是这样的，就像 bootstrap 为后端程序员提供了一个样式框架可以让他们不去纠结样式问题，BaaS 为前端程序员提供了一个服务平台，可以不需要掌握后端技术就可以对数据进行操作，实现对数据的增删改查操作。

这样的平台有很多：AppCan、Firebase、StackMob、End.js、LeanCloud 等，我用 LeanCloud 来体验一下这样的服务平台。

二、LeanCloud

1、什么是 LeanCloud

LeanCloud 是国内的移动应用一站式云服务。LeanCloud 提供了数据存储、实时消息、统计分析以及多种扩展组件，全面涵盖移动应用开发的需求，支持 iOS、Android、Web 等多平台。它帮助开发者摆脱后端开发负担以专注于产品创新，同时缩短开发周期、节省开发投入、快速进入市场。
