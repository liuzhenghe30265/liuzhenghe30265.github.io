发表于 2019-07-01 | 分类于 JavaScript高级程序设计笔记

一、Global 对象

没有全局变量或全局函数，所有在全局作用域中定义的属性和函数，都是 Global 对象的属性。

1.1、URI（Uniform Resource Identifiers，通用资源标识符）编码方法：

  • encodeURI()
  • encodeURIComponent()

encodeURI() 和 encodeURIComponent() 两个 URI 编码方法可以对 URI 进行编码，他们用特殊的 UTF-8 替换所有无效的字符。区别就是：encodeURI() 只会替换空格，其他字符不变，而 encodeURIComponent() 会替换例如反斜杠、冒号、问号、空格这些非标准字符。

与这两个编码方法对应的解码方法是 decodeURI() 和 decodeURIComponent()：

同样，decodeURI() 也只会将空格解码，其他字符不变，而 decodeURIComponent() 将所有的特殊字符都替换成了原来的字符。

1.3、Global 对象的属性

特殊值 undefined、特殊值 NaN、特殊值 Infinity、构造函数 Object、构造函数 Array、构造函数 Function、构造函数 Boolean、构造函数 Numner、构造函数 String、构造函数 Regexp、构造函数 Date 等都是 Global 对象的属性。
