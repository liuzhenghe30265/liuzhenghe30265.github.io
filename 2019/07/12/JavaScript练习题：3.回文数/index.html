一、题目

两数之和。

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那两个整数，并返回他们的数组下标。

示例：

  • 给定 nums = [2, 7, 11, 15], target = 9；
  • 因为 nums[0] + nums[1] = 2 + 7 = 9；
  • 所以返回 [0, 1]。

三、知识点

ES6 新增的数据类型 Map 和 Set。

  • Map
  • Set

3.1、Map

ES6 引入了新的数据类型 Map，Map 是一组键值对的结构，具有极快的查找速度，例如要根据同学的名字查找成绩，如果用 Array 就需要两个 Array，这种方法，如果 Array 越长，那么耗时也就越长。

如果使用 Map 的话只需要一个“名字-成绩”对照表，直接根据名字查找成绩，无论这个表有多大，查找速度都不会慢：

Map 具有以下方法：

  • set() 添加新的键-值，如果键存在，则修改值；
  • has() 根据键判断是否存在；
  • get() 根据键获取值；
  • delete() 根据键删除对应的键值对。

3.2、Set

和 Map 类似，也是一组 key 的集合，但不存储 value，由于 key 不能重复，所以 Set 中没有重复的 key。
