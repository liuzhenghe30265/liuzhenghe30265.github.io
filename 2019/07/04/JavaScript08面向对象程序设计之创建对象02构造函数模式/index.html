二、构造函数模式

按照惯例，构造函数的函数名首字母应该大写，而非构造函数的函数名首字母应该是小写。构造函数中没有显式的创建对象，直接将属性和方法赋值给了 this 对象。在创建新实例的时候使用了 new 操作符。调用构造函数，实际上会经历4个步骤：

  1. 创建一个新的实例对象；
  2. 将构造函数的作用域赋值给新对象（this 就指向了这个新对象）；
  3. 执行构造函数中的代码（为新对象添加属性和方法）；
  4. 返回一个新的实例对象。
