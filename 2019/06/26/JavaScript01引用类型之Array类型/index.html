JavaScript01引用类型之Array类型

发表于 2019-06-26 | 分类于 JavaScript高级程序设计笔记

一、数组检测

二、转换方法

  • toLocaleString() 把数组转换为字符串。
  • toString() 把数组转换为字符串。
  • valueOf() 返回的还是数组。

三、栈方法

后进先出的数据结构（LIFO）。

  • push() 向数组末尾添加元素。
  • pop() 从数组末尾删除元素。
    他们都会改变原数组。

四、队列方法

先进先出的数据结构（FIFO）。

  • shift() 删除数组的第一项。
  • unshift() 向数组的前面添加元素。
    会改变原数组。

五、重排序方法

  • reverse() 反转数组。
  • sort() 数组排序（默认是按照 ASCII 来排序的）。

显然 sort() 方法并没有按照数字从小到大来排序，因为他是按照 ASCII 来排序的。

六、操作方法

  • concat() 他不会改变原数组，而是复制出一份数组，在这个数组的基础上再添加元素。
  • slice() 截取起始位置间的元素，不会改变原数据。如果不传第二个参数，也就是不传终止位置，截取的就是从起始位置到数组最后。
  • splice() 删除、插入、替换。他会改变原数组。

删除：传入两个参数，删除开始的位置和删除结束的位置，返回剩下的元素。

插入：传入三个参数，开始插入的位置，要删除的项（如果不删除原数组，就为 0）和要插入的元素。

替换：传入三个参数，替换开始的位置，要替换的项（删掉这些项）和要替换的目标元素。

七、数组位置方法

  • indexOf() 传入元素，返回数组中第一次出现该元素的位置，如果元素不存在，则返回 -1。
  • lastIndexOf() 传入元素，从数组末尾往前找，同样返回数组中第一次出现该元素的位置。
    indexOf() 和 lastIndexOf() 如果传入两个参数，第一个参数是要查找位置的元素，第二个参数是从哪个位置开始查找。

八、数组迭代方法

ECMAScript5 为数组定义了 5 个迭代方法：

every()、some()、map()、forEach()、filter()，每个方法都接收两个参数，一个是要在每一项上运行的函数，一个是运行该函数的作用域对象。传入这些方法中的函数都接收三个参数，数组项的值，该数组项在数组中的位置和数组对象本身（使用 fliter 实现数组去重时使用到了这三个参数）。

8.1、every

对数组中每一项运行给定函数，如果有一项不满足条件，就返回 false，都满足返回 true。

8.2、some

对数组每一项运行给定函数，只要有一项满足条件，就返回 true。

8.3、map

对数组每一项运行给定函数，返回函数每次调用的结果组成的数组。

8.4、forEach

对数组每一项运行给定函数，这个方法没有返回值。

8.5、filter

对数组每一项运行给定函数，返回满足条件的项组成的数组。

实现一个数组去重：根据 indexOf() 总返回数组中元素第一次出现的位置的特性，将数组中第一次出现的这些元素返回。

九、归并方法

ECMAScript5 还为数组提供了两个归并方法：reduce() 和 reduceRight()，
这两个方法都会迭代数组的每一项，然后构建一个最终返回的值。他俩传入的函数都可以接收 4 个参数。
x 和 y 分别代表前一项和后一项。
